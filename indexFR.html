Mes projets

Le robot 'SKY_BOT'

Le SKY BOT est un projet réalisé par des étudiants de deuxième année au Lycée Louis Armand. Ce robot peut être contrôlé à l'aide d'un joystick, d'un site web et même par la voix, en plus de répondre à nos questions.

Compétences utilisées : C++, JavaScript, HTML, CSS sur Raspberry Pi et ESP32. Le code

Jeu vidéo 'Player RUN'

Le jeu "Player run" est une création personnelle que j'ai développée sur Unity. C'est un jeu d'action-aventure en 2D dans lequel le joueur incarne un personnage qui doit survivre contre des ennemis et trouver une sortie d'une grotte.

Compétences utilisées : C# et Unity.

Gestion de Stock

Mise en place d'un stock automatisé en temps réel via une IHM pour l'entreprise Iris Systeme. L'utilisateur peut visualiser les produits présents dans le stock, en ajouter facilement en créant un QR Code avec les informations du produit à coller dessus, et retirer des produits en scannant les QR Codes. Les informations des produits sont stockées dans une base de données MariaDB.

Compétences utilisées : JavaScript avec NodeJS et EletronJS, HTML, CSS avec TailwindCSS, SQL sur MariaDB, PHPMyAdmin, Grafana. Le code
